grid

no lineboxes or flexboxes you can create any layout that you like, as long as you like grids
like a flex container, we create a grid container by giving an element display: grid any children of the element now become grid items we set up our grid by defining rows and colums then, when we place children inside the container, they're laid out according to the grid
